The Problem

I wanted real practice with Angular's component/service/routing architecture on something bigger than a single-page tutorial demo — a small app with a realistic-feeling dataset and full CRUD operations against it, not just a static list.

My Role

Solo project. I built a restaurant-themed app in Angular, with components for listing, adding, editing, and deleting menu items, backed by a json-server instance acting as a mock REST API and data layer.

Outcome
